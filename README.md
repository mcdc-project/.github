# MC/DC Organization Configuration

![MC/DC logo](https://raw.githubusercontent.com/mcdc-project/mcdc/main/assets/mcdc-logo.svg)

[![License](https://img.shields.io/badge/License-BSD_3--Clause-blue.svg)](https://opensource.org/licenses/BSD-3-Clause)

This repository contains shared GitHub configuration for the [MC/DC organization](https://github.com/mcdc-project).

Its purpose is to centralize resources that are reused across repositories, such as:

* Organization-wide GitHub labels
* GitHub Actions workflows
* Issue and pull request templates
* Community health files
* Other shared GitHub configuration

Individual repositories (e.g., [mcdc](https://github.com/mcdc-project/mcdc) and [mcdc-vvp](https://github.com/mcdc-project/mcdc-vvp)) remain responsible for their own repository-specific configuration and labels.
